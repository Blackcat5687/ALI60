# English Talk 🎙️✨

**English Talk** هو تطبيق أندرويد متكامل ومصمم بأسلوب عصري ليكون مُعلمك الشخصي والذكي لإتقان اللغة الإنجليزية من خلال المحادثات الصوتية والتفاعلية المباشرة المدعومة بالذكاء الاصطناعي (Gemini API).

---

## 🎨 المميزات الرئيسية
- **محادثات تفاعلية حية (Dialogue Practice):** تواصل وتحدث مع معلمك الذكي بلغة طبيعية وسلسة.
- **تقييم دقيق للنطق (Pronunciation Feedback):** احصل على مراجعة صوتية فورية مع نسبة مئوية لدقة نطق الكلمات والعبارات.
- **تسجيل دخول حقيقي وآمن (Real Google Authentication):** إمكانية تسجيل الدخول السلس والآمن عبر حساب Google وحفظ صورتك الشخصية وبياناتك ومزامنتها.
- **قاعدة بيانات محلية (Room Database):** لحفظ تقدمك ومستواك التعليمي والدرجات وقائمة الأخطاء والكلمات الصعبة للمراجعة لاحقاً دون فقدان أي بيانات.
- **واجهة مستخدم Material 3 عصرية:** واجهة مريحة وجذابة تدعم الوضعين الداكن والفاتح بالكامل وتنسيق ألوان ديناميكي متناسق مع حجم الشاشة والأجهزة اللوحية.

---

## 🛠️ التقنيات المستخدمة (Tech Stack)
- **لغة البرمجة:** Kotlin (100%)
- **واجهة المستخدم:** Jetpack Compose (مع دعم Edge-to-Edge و Window Size Classes للأجهزة اللوحية والمطوية)
- **هندسة البرمجيات:** MVVM (Model-View-ViewModel) مع Clean Architecture
- **قاعدة البيانات:** Room Database للحفظ المحلي والمزامنة المستمرة.
- **الذكاء الاصطناعي:** دمج مباشر مع Gemini API لتقديم ردود ذكية ومخصصة وتصحيح نطق المستخدم.
- **المصادقة:** Firebase Authentication مع Google Sign-In و Jetpack Credential Manager.
- **تحميل الصور:** Coil لسرعة وكفاءة عرض الصور والوجوه الرمزية (Avatars) وصور المستخدمين.

## 🚀 كيفية التشغيل وبناء المشروع

المشروع جاهز ومُهيأ بالكامل للبناء الفوري على أي نظام تشغيل، وكذلك في بيئات التطوير المستمر مثل **GitHub Actions** باستخدام **JDK 17** أو أحدث.

### 1. البناء السريع عبر الطرفية (Terminal):
لتشغيل عملية البناء التلقائية وتوليد ملف الـ APK التجريبي، افتح الطرفية في المجلد الرئيسي للمشروع ونفذ الأوامر التالية:

**على أنظمة Linux / macOS / Ubuntu:**
```bash
chmod +x gradlew
./gradlew assembleDebug
```

**على نظام Windows (Command Prompt / PowerShell):**
```powershell
.\gradlew.bat assembleDebug
```

بعد انتهاء البناء بنجاح، ستجد ملف الـ APK الناتج في المسار التالي:
`app/build/outputs/apk/debug/app-debug.apk`

### 2. الفتح عبر Android Studio:
1. افتح برنامج **Android Studio** (نسخة Ladybug أو أحدث موصى بها).
2. اختر **Open** ثم حدد المجلد الرئيسي للمشروع.
3. انتظر ريثما ينتهي البرنامج من مزامنة ملفات Gradle تلقائياً (Gradle Sync).
4. اضغط على زر **Run** لتشغيل التطبيق على محاكي أو جهاز حقيقي مرتبط.

## 🔑 إعدادات البيئة ومفاتيح الـ API
يدعم التطبيق حقن وحفظ مفاتيح الـ API بطريقة آمنة ومرنة:
1. **مفتاح Gemini API:** يمكنك إدخال وتفعيل المفتاح مباشرة من شاشة الإعدادات داخل التطبيق بكل أمان للتجربة الفورية.
2. **صلاحية المايكروفون:** يُطلب الإذن تلقائياً من المستخدم للتسجيل الصوتي لإرسال الصوت وتحليله بواسطة الذكاء الاصطناعي.
